import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

We've simplified layout components to make them easier to use. There is a few things you need to update in order to use new layout.

### Mandatory changes

<!-- 1. Change `<LayoutContent/>` to `<Layout/>`.  -->
1. Rename props `<Layout/>`:
	- `topStart` to `sidebarHeader`
	- `topEnd` to `sidebarFooter`
	- `sideBar` to `navigation`

<Tabs
  defaultValue="before"
  values={[
    { label: "Before", value: "before" },
    { label: "After", value: "after" },
  ]}
>
  <TabItem value="before">

```tsx title="Step 1"
<Layout
	topStart={<Heading label={heading} switchers={switchers} />}
	topEnd={<Userbox />}
	sideBar={SideMenu && <Sidebar SideMenu={SideMenu} />}
/>
```

  </TabItem>
  <TabItem value="after">

```tsx title="Step 1"
<Layout
	sidebarHeader={<Heading label={heading} switchers={switchers} />}
	sidebarFooter={<Userbox />}
	navigation={SideMenu && <Sidebar SideMenu={SideMenu} />}
/>
```

  </TabItem>

</Tabs>

2. Move prop `actions` from `<TitleBar/>` to page component (e.g. `<Page/>`, `<GenericPage/>`, `<LayoutPage/>`).
3. Remove `<TitleBar/>` component and content of `<TitleBar/>` insert into prop `title` on page component.

<Tabs
  defaultValue="before"
  values={[
    { label: "Before", value: "before" },
    { label: "After", value: "after" },
  ]}
>
  <TabItem value="before">

```tsx title="Steps 2 and 3"
<GenericPage 
	pageName="blogList"
>
	<TitleBar actions={<PageLinkButton to="blogCreate">Add article</PageLinkButton>}>Article</TitleBar>
	...
</GenericPage>
```

  </TabItem>
  <TabItem value="after">

```tsx title="Steps 2 and 3"
<GenericPage 
	pageName="blogList"
	title="Article"
	actions={<PageLinkButton to="blogCreate">Add article</PageLinkButton>}
>
	...
</GenericPage>
```

  </TabItem>

</Tabs>

4. Change `<LayoutInner/>` to `<LayoutPage/>`.
5. Remove `<LayoutSide />` component and content insert into prop `side` on `<LayoutPage />`.

<Tabs
  defaultValue="before"
  values={[
    { label: "Before", value: "before" },
    { label: "After", value: "after" },
  ]}
>
  <TabItem value="before">

```tsx title="Steps 4, 5"
<LayoutContent>
	<LayoutInner>
	...
	</LayoutInner>
	<LayoutSide>
		<PersistButton />
	</LayoutSide>
</LayoutContent>
```

  </TabItem>
  <TabItem value="after">

```tsx title="Steps 4, 5"
<LayoutContent>
	<LayoutPage side={<PersistButton />}>
	...
	</LayoutPage>
</LayoutContent>
```

  </TabItem>

</Tabs>

6. We are not supporting `<AltPersistButton/>` anymore. Remove it or change it to `<PersistButton/>`.
7. Change `<GenericPage/>` which wraps tenant pages to `<Page/>`.

### Recomended changes

Remove unnecessary `<Box/>` component. There is new component `<Section/>` which should be used instead of `<Box/>` for defining groups of fields.
