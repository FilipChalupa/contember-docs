Customizing admin distinguishes between customizing the **theme** (color) and **scheme** (dark, light). Theme sets color independently on the scheme. Scheme defines light or dark modes for elements.

## Color themes

## Color schemes

Contember combine theme for content and controls with light and dark scheme resulting in many flexible combinations. By default, the content is themed with **default** theme and controls in **primary** theme.
