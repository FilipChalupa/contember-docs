### Text field

The `TextField` component renders a text field. The text field is used for basic string editing and has no rich text editing capabilities.

```tsx
<TextField field="name" label="Name" />
```

#### Props

### Rich text field

The `RichTextField` component renders a rich text field. The text field is used for rich text editing.

<Tabs
  defaultValue="admin-simple"
  values={[
    { label: "Component - simple", value: "admin-simple" },
    { label: "Component - advanced", value: "admin-advanced" },
    { label: "API schema", value: "schema" },
  ]}
>
  <TabItem value="admin-simple">

```tsx
<RichTextField field="content" label="Content" />
```

  </TabItem>
  <TabItem value="admin-advanced">

```tsx
<RichTextField
  field="content"
  label="Content"
  inlineButtons={[
    [
      RichEditor.buttons.italic,
      RichEditor.buttons.underline,
      RichEditor.buttons.strikeThrough,
    ],
  ]}
/>
```

Component displays only **italic**, **underline** and **strike through** buttons in format text option.

  </TabItem>

  <TabItem value="schema">

```tsx

  content = def.stringColumn()
}
```

  </TabItem>
</Tabs>

### Field View

The `FieldView` component renders a field value.

```tsx

  field="startAt"
  label="Event start"
  render={(filedAccessor) => <>{fieldAccessor.value}</>}
/>
```
