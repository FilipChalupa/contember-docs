## Block editor

The `BlockEditor` component is the main component of the editor. It is responsible for rendering the editor's toolbar and the editor's content.

```tsx
<BlockEditor
    contentField="json"
    field="content.blocks"
    label="Content"
    sortableBy="order"
/>
```
